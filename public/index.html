Olá, somos alunos do curso de Ciência da Computação e felizmente estamos concluindo esse curso maravilhoso, para tanto precisaremos terminar o Trabalho de Conclusão de Curso. Nosso projeto consiste em desenvolver um sistema com o objetivo de reproduzir músicas em um ambiente compartilhado e fugir das tradicionais "músicas de elevador". Para isso o sistema deverá ser capaz de analisar os indivíduos que compõem esse grupo e reproduzir músicas que os agradem, analisando sempre os gostos de cada indivíduo e tentando traçar a maior intercessão possível dentre as características presentes. Entretanto para isso necessitamos de dados reais para que possamos desenvolver e testar o sistema, e é nesse parte você poderá nos ajudar!


Utilizaremos a base de dados do Spotify, mas para isso precisamos da permissão dos usuários. Ou seja:



Não se preocupe que os dados serão coletados anonimamente! A menos que você queira participar mais intensamente do desenvolvimento do sistema, nos ajudando com os testes do sistema, para isso será necessário somente o seu e-mail.


Para fins de transparência listamos abaixo as permissões que serão utilizadas.

Permissões necessárias

  • user-top-read : Acessar seus artistas e músicas mais escutados.
  • user-read-recent-played : Acessar o que você escutou recentemente.
  • user-library-read : Acessar suas faixas, álbuns e programas salvos.
  • playlist-read-private : Acessar suas playlists particulares.
  • playlist-read-collaborative : Acessar suas playlists colaborativas.

Permissões opcionais

  • user-read-email : Obter seu endereço de e-mail.
  • user-read-birthdate : Receber sua data de nascimento.

Importante: As permissões só permitem ler os dados, nenhuma informação poderá ser alterada ou removida. Segue o link do Spotify para mais informações.
